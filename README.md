# 폴리곤 아트 제너레이터 (Polygon Art Generator)

> 이 프로젝트는 **Google AI Studio**를 통해 생성 및 개선되었습니다.

업로드한 이미지를 분석하여 감각적인 폴리곤 아트(Polygon Art)로 변환해주는 웹 애플리케이션입니다. 모든 이미지 처리 및 변환 과정은 서버 전송 없이 사용자의 브라우저 내에서 안전하게 이루어집니다.

## ✨ 주요 기능 및 특징

* **적응형 격자 (Adaptive Grid / Quadtree) 알고리즘:**
  * 단순한 무작위 점 추출이 아닌, 이미지를 격자(Grid)로 나누어 각 영역에서 가장 뚜렷한 윤곽선을 찾아냅니다.
  * 눈, 코, 입 등 디테일이 중요한 영역(Salient Gradient)을 감지하면 자동으로 격자를 더 잘게 쪼개어(Subdivide) 중요한 이목구비가 뭉개지지 않도록 보호합니다.
  * 결과적으로 사람이 직접 면을 나눈 듯한 **구조적이고 깔끔한 일러스트 느낌**을 줍니다.

* **델로네 삼각 분할 정제 (Delaunay Refinement):**
  * 바늘처럼 지나치게 뾰족하고 기형적인 삼각형(Sliver triangles)이나 너무 거대한 삼각형을 감지합니다.
  * 해당 삼각형의 가장 긴 변을 분할하여 전체적으로 안정감 있고 균형 잡힌 폴리곤 구조를 완성합니다.

* **복원 품질 (Reconstruction Quality) 세밀 조정:**
  * 0%부터 100%까지 슬라이더를 통해 폴리곤의 밀도를 실시간으로 조절할 수 있습니다.
  * **100%:** 사전 블러(Blur) 제거, 촘촘한 격자, 높은 윤곽선 민감도를 통해 픽셀 레벨에 가까운 정교한 복원을 수행합니다.
  * **0%:** 강한 블러 적용, 넓은 격자를 통해 자잘한 디테일을 무시하고 큼직하고 추상적인 진정한 로우폴리(Low-poly) 아트를 생성합니다.

* **시각적 애니메이션 및 속도 조절:**
  * 원본 이미지 -> 윤곽선 추출(Sobel Filter) -> 특징점 추출 -> 델로네 삼각 분할 -> 색상 채우기 -> 최종 렌더링의 전체 과정을 시각적으로 감상할 수 있습니다.
  * 삼각형들이 화면 곳곳에서 무작위로 튀어나오는(Popping) 유기적인 트랜지션 효과를 적용했습니다.
  * **재생 속도 슬라이더**를 통해 애니메이션 속도를 0.5배속에서 4.0배속까지 자유롭게 조절할 수 있습니다.

* **자연스러운 색상 추출 (Median Color Sampling):**
  * 각 삼각형의 색상을 결정할 때 단순 산술 평균이 아닌 내부 픽셀들의 중앙값(Median)을 샘플링합니다. 이를 통해 이미지의 노이즈나 튀는 색상(Popping color)을 방지하고 훨씬 부드럽고 조화로운 결과물을 만들어냅니다.

## 🛠 기술 스택
* React 18
* TypeScript
* Tailwind CSS
* Vite
* Delaunator (빠른 2D 델로네 삼각 분할 라이브러리)
* HTML5 Canvas API (이미지 처리 및 렌더링)
